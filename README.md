# AI_Investment 项目总览

## 项目简介
AI_Investment 是一个集成了多元化投资理念、基本面分析、资产配置、自动化工具与前端展示的综合性投资研究与实践平台。项目包含量化分析、自动化脚本、前端可视化、文档知识库等多个模块，旨在提升投资决策的科学性与效率。

## 目录结构说明
- `finguard/`：核心后端服务与量化分析引擎，包含 C++ 源码、配置、测试等
- `frontend/`：前端展示与交互，基于现代 Web 技术
- `workbook/`：项目经验、自动化、文档规范等知识沉淀
- `优化/`：投资策略、基本面分析、资产配置等优化文档
- `学习文档/`：学习笔记、技术积累、面试资料等
- `skills/`：AI agent 相关技能与自动化脚本
- `tools/`：辅助工具与脚本
- `config/`：全局配置文件
- 其他：各类项目文档、阶段计划、面试手册等

## 快速开始
1. 克隆仓库：
   ```bash
   git clone https://github.com/NeoChenjf/Finguard.git
   ```
2. 后端构建与运行（以 finguard 为例）：
   - 进入 `finguard/` 目录，参考 `README.md` 或 `CMakeLists.txt` 进行编译
3. 前端启动：
   - 进入 `frontend/` 目录，执行 `npm install && npm run dev`
4. 详细文档与规范请参见 `workbook/`、`优化/`、`学习文档/` 等目录

## 主要功能
- 基本面分析与自动化数据采集
